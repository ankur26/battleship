# Battleship

This is an implementation for Battleship game [Link](https://en.wikipedia.org/wiki/Battleship_(game)) from the The Odin Project's Javascript series [Link](https://www.theodinproject.com), this project particularly emphasizes using Test Driven Development methodology.


## Concepts learnt before starting the project.

1. Testing and it's importance. 
2. Unit testing, pure functions and side effects in functions
3. TDD and the red-green-refactor cycle
4. Testing framework Jest.
5. Mocking side effects and why mocking is needed.
